# Notes App
A simple ReactJS App using:
  * React Hooks
  * Router-Dom
  * Consuming backend API's
  * MaterialUI
  * Bootstrap

## How To Use
Install dependencies:

`$ npm install`

Run application:

`$ npm start`

The app will start running at http://localhost:3000.
